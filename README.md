## Description
This project contains Playwright automation tests for Assignment 1 of IT3040.
The tests validate Singlish to Sinhala conversion behavior of the Swift Translator web application.

## Tools Used
- Playwright
- Node.js
- VS Code

## How to Run Tests
1. Install dependencies
   npm install

2. Install Playwright browsers
   npx playwright install

3. Run Playwright tests
   npx playwright test

4. View HTML report (optional)
   npx playwright show-report
